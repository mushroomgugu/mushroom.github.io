201.数字范围按位与

Posted on 2021-01-26 In leetcode , 题解

题目

给定范围 [m, n]，其中 0 <= m <= n <= 2147483647，返回此范围内所有数字的按位与（包含 m, n 两端点）。

示例1

示例2

类别

位运算

题解

位移

将两端的数右移，找到公共前缀，统计移动次数，得到小端大端公共前缀后面置为0的结果。

Brian Kernighan 算法

用于清除二进制串中最右边的1

方法：

n&(n-1)
